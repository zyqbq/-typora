# 作用域

**变量在某范围内的起作用和效果**

**减少命名冲突**

### **全局作用域**

script标签内或者，单独的js文件中

全局变量，在全局都能使用

比较占内存

```js
var num=1
```

### 局部作用域（函数作用域）

只能在函数内部使用

函数的形参也可以看做局部变量

执行就销毁，节约内存

```js
function(arg){
    //局部作用域
    var num=20；
    //这种也是全局变量
    num2=30；
}
```

块级作用域



### 作用域链

由内到外一层一层

就近原则
